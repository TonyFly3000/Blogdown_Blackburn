3.4.1 dplyr

dplyr is a grammar of data manipulation create by Hadley Wickham, providing a consistent set of verbs that help you solve the most common data manipulation challenges:

Filter row by condidtion[key word:filter]:

mtcars%>%filter(cyl==6)                    # filter cly==6
mtcars%>%filter(cyl==6 & (hp >120|hp<110)) # filter cly==6 and (hp >120 or hp<110)

Filter row by row number:

# get 3,4,5 rows
mtcars[3:5,]

select column[key word:select]:

mtcars%>%select(mpg,cyl)                    # select mpg and cyl column
mtcars%>%select(-mpg)                       # select all but exclude mpg

select column by column number:

# get 3,4,5 columns
mtcars[,3:5]

group by,create variable[key word:group_by,summarise]:

#number and distinct cyl number
mtcars%>%group_by()%>%summarise(row_number=n(),cyl_number=n_distinct(cyl))   # sum row

group by,create variable and then base on the resutle to create a new variable[key word:group_by,summarise,mutate]:

# sum mpg and mean mpg for each cyl and create a new variable mpg_sum_k
mtcars%>%group_by(cyl)%>%summarise(mpg_sum=sum(mpg),mpg_mean=mean(mpg)%>%mutate(mpg_sum_k=mpg_sum/1000)

order and transpose[key word:arrange,t]:

mtcars%>%arrange(desc(cyl))   # order by cyl decreasing
t(mtcars)                     # transpose

join[key word:left_join,full_join,inner_join]:

left_join(mtcars[c(1:3),],mtcars[c(3:5),],by=c('mpg'='mpg'))   # left join 

full_join(mtcars[c(1:3),],mtcars[c(3:5),],by=c('mpg'='mpg'))   # full join 

inner_join(mtcars[c(1:3),],mtcars[c(3:5),],by=c('mpg'='mpg'))  # inner join

Instead of creating a lot of dummy data set,Pipe operator (%>%) allow you to do multiple action on one dataset and output into a new dataset

# first select and then filter and then summarize
new_mtcars =mtcars%>%filter(cyl==6)%>%select(mpg,cyl)%>%group_by(cyl)%>%summarise(total_mpg=sum(mpg))
